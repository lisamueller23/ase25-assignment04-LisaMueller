# Non-Functional Requirements

* As a customer, I want the platform to be responsive so that I can quickly search for meals and order ingredients.
* As a customer, I want the platform to be lightweight so that I can quickly browse and order meals.
* As a customer, I want the platform to be intuitive to use so that I can easily navigate.
* As a developer, I want the platform to be easy to maintain so that I can quickly make adjustments.
* As a customer, I want the platform to extend with new meal so that I have even more choice.
* As a customer, I want the platform to extend dietary rules so that I have even more information about the meal and ingredients.
* As a business owner, I want the platform to not lose order details if the system crashs so that all customers get their orders.
