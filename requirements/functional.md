# Functional Requirements

* As a customer, I want to browse meals offered by the catering service so that I can see what options are available to order.
* As a customer, I want to view the list of ingredients required for each meal so that I know that it only contains ingredients that I like.
* As a customer, I want to order ingredients for a selected meal so that I have all the ingredients I need for that dish.
* As a customer, I want to schedule ingredients deliveries so that I can receive the ingredients when I need them.
* As a customer, I want to track the status of my orders so that I know when the order will arrive.
* As a customer, I want to search meals by name, ingredient or category so that I can find meals so that I can find meals that suit my preferences.
