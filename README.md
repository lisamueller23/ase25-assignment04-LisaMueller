## How to Run the Demo

### Prerequisites
- Python 3 installed on your system (check with `python3 --version`).
- A JSON file containing meals and their ingredients (example provided in `sample_data/meal-ingredients.json`).

### Steps
1. Clone or download this repository.
2. Ensure the JSON file `meal-ingredients.json` is located in the folder `sample_data/`.
3. Open a terminal and navigate to the project folder.
4. Run the program with:
   python3 smart_carter.py (with IOS System)
